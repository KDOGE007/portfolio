Movie Search Web App (React)

A React personal project where the user can search a movie title and look for its' rating and description. This basic web app parse the user input into a request and send it to TMDb, www.themoviedb.org, through an API and render the returned result in the same page below. This is achieved by using React Hooks, useState.

TODO App

Everyone's favourite project, a Todo App. The user types their todo item and it will populate the item on the same page once an enter key is pressed. They can cross the items and also delete them from the list. The list will persist even if the user refreshes or navigates away from the page. This was built with a vanilla JS with ES6 syntax wherever I can. The persists state was achieved by storing the items and its key in a localstorage area and recall them when the web page is re-rendered.
